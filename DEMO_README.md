## 🚀 How to Run

### Option 1: Using the wrapper script (recommended)
```bash
./run_demo.sh
```

### Option 2: Direct execution
```bash
python3 demo_agent_registry.py
```

### Option 3: Make executable and run
```bash
chmod +x demo_agent_registry.py
./demo_agent_registry.py
```
